# Krono — Funcionalidades e Comportamentos

## O que é
Cronômetro Android que flutua sobre outros apps via `WindowManager` overlay. O usuário mede tempo sem sair do app que está usando.

---

## Funcionalidades Principais

### Overlay Flutuante
- Janela `WindowManager` com `TYPE_APPLICATION_OVERLAY`
- Arraste livre + snap automático nas bordas
- Escala ajustável (0.5x a 1.5x), cores, opacidade e cantos configuráveis
- Botões visíveis: Play/Pause, Reset, Fechar
- **Gestos:** 1 toque = play/pause · 2 toques = reset · long press = menu rápido

### Menu Rápido (Long Press no Overlay)
- Expande verticalmente abaixo do overlay com animação spring
- 3 ícones: ⚙️ Configurações · 🎯 Toggle Modo Foco · 💡 Toggle Manter Tela Ligada
- Fecha automaticamente após 5s de inatividade ou toque no corpo do overlay
- Enquanto aberto: remove `FLAG_NOT_FOCUSABLE` para receber eventos corretamente

### Modo Foco
- Ativa automaticamente ao dar Play se `focusModeEnabled = true` e overlay visível
- `FocusActivity` cobre a tela de preto
- Toque na área preta → oculta por 6s → reativa automaticamente
- `FLAG_KEEP_SCREEN_ON` forçado independente de outras configurações
- Persiste no Pause e no Reset (não é desativado por eles)
- Desativa apenas quando o overlay é fechado (`ACTION_FOCUS_DISMISSED`)

### Manter Tela Ligada
- `FLAG_KEEP_SCREEN_ON` + `WakeLock` no overlay
- Funciona independente do cronômetro estar rodando ou pausado

### Abrir Diretamente (autoLaunch)
- `LaunchProxyActivity` como ponto de entrada do ícone do launcher
- Se `autoLaunch = true` e permissão OK: inicia serviço e encerra sem mostrar o app
- Se não: abre `MainActivity` normalmente

### Notificação Persistente
- Foreground Service com cronômetro nativo Android (zero latência)
- Botões: Play/Pause, Reset, Fechar
- Após reset: notificação atualiza para "00:00:00" imediatamente
- Quando pausado/zerado: exibe tempo congelado como texto

### Tempo Limite
- Configurável em HHHH:MM:SS
- Encerra o serviço automaticamente ao atingir

### Ciclo de Doação
- Acumula tempo de uso no DataStore
- A cada 12h acumuladas: esconde overlay, abre `DonationDialog`
- Após interação: volta ao overlay

### Temas
4 temas disponíveis + Auto (segue sistema):
- `DarkModern` · `SolarizedDark` · `LightModern` · `SolarizedLight`
- Selecionável em Configurações → Aparência
- Aplicado tanto na `MainActivity` quanto no overlay (`ComposeView` do `MainService`)

### Atualização Automática
- Verifica GitHub Releases 1x/dia (em background, sem travar o app)
- `ChangelogDialog`: mostra novidades da versão instalada (arquivo local `res/raw/changelog.md`)
- Botão "Verificar Atualizações" consulta o GitHub na hora
- Se há nova versão: `UpdateDialog` com changelog e botão de download
- Download via `DownloadManager` com progresso em tempo real
- Instalação automática via `FileProvider` + Intent do instalador nativo
- APK deletado automaticamente após instalação via `BroadcastReceiver`

---

## Permissões Necessárias
- `SYSTEM_ALERT_WINDOW` — overlay sobre outros apps
- `FOREGROUND_SERVICE` — serviço em background
- `POST_NOTIFICATIONS` — notificação persistente (Android 13+)
- `REQUEST_INSTALL_PACKAGES` — instalação de APK de atualização
- `INTERNET` — verificação de atualizações
- `RECEIVE_BOOT_COMPLETED` — reiniciar serviço após reboot
