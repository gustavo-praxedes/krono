# Krono — Stack e Configurações Técnicas

## Linguagem e SDK
- **Kotlin** — 100% (sem código Java).
- `minSdk` 26 (Android 8.0) · `targetSdk` 35 (Android 15) · `compileSdk` 35.

## UI e Design
- **Jetpack Compose** exclusivo.
- **Material 3** (`androidx.compose.material3`).
- **Skydoves ColorPicker** para seleção de cores customizadas.
- **Temas Customizados**: Gerenciados via `KronoTheme.kt` com suporte a 4 paletas pré-definidas + cores personalizáveis pelo usuário.

## Programação Assíncrona
- **Kotlin Coroutines** com `StateFlow` e `SharedFlow`.
- `collectAsState()` nos Composables para reatividade de UI.
- `collectLatest` em observers críticos no `MainService`.

## Persistência de Dados
| Classe | Mecanismo | Papel |
|---|---|---|
| `OverlayDataStore.kt` | DataStore Preferences | Configurações do overlay (cores, posição, escala). |
| `TimerPreferences.kt` | SharedPreferences | Persistência síncrona para tempo acumulado e estado pós-boot. |
| `OverlayConfig.kt` | Data Class | Representação imutável das configurações visuais. |
| `TimeState.kt` | Data Class | Representação imutável do estado do cronômetro. |

## Navegação
- `androidx.navigation.compose`: Gerencia rotas internas (`TIMER`, `SETTINGS`).

## Background e Overlay
- **MainService**: Foreground Service de alta prioridade.
- **WindowManager**: Uso de `TYPE_APPLICATION_OVERLAY`.
- **Flags de Foco**: `FLAG_NOT_FOCUSABLE` (padrão) vs foco dinâmico para menus e inputs.

## Atualização e Distribuição
- **UpdateChecker.kt**: Integração com GitHub API (`releases/latest`).
- **ApkInstaller.kt**: Download via `DownloadManager` e instalação via `FileProvider`.
- **FileProvider**: Authorities: `${applicationId}.provider`. Cache de APKs em `cacheDir`.

## Configurações de Build (Gradle)
- **VersionCode**: Gerado dinamicamente no formato `yyMMddHH`.
- **VersionName**: Sincronizado com `package.json` (SemVer).
- **Assinatura**: Configurações de Keystore carregadas de `keystore.properties` (local) ou env vars (CI/CD).
- **ProGuard**: Habilitado em release para redução de tamanho e ofuscação básica.
