# ⏱️ Cronômetro Flutuante (Floating Timer)

Um aplicativo Android leve e prático que mantém um cronômetro sempre visível sobre outros aplicativos. Ideal para quem precisa monitorar o tempo enquanto estuda, trabalha ou realiza tarefas multitarefa no celular.

---

## 🚀 Funcionalidades Principais

* **Sempre no Topo:** O cronômetro flutua sobre qualquer app, permitindo que você acompanhe o tempo sem precisar alternar janelas.
* **Customização Visual:** Ajuste o tamanho (escala) e as cores do cronômetro para que ele se adapte ao seu estilo ou papel de parede.
* **Controle de Produtividade:** Sistema de monitoramento de uso contínuo que incentiva pausas saudáveis.
* **Interface Minimalista:** Design focado no que importa, sem distrações.
git
## 🛠️ Tecnologias
Este projeto utiliza as tecnologias mais modernas do ecossistema Android:
* **Kotlin** & **Jetpack Compose** (Interface moderna e declarativa).
* **Android Services:** Para manter o cronômetro ativo mesmo fora do app.
* **SharedPreferences:** Para salvar suas preferências de cores e tamanho.

## 📦 Como Instalar
1. Acesse a aba [Releases](https://github.com/gustavo-praxedes/cronometro-flutuante/releases) deste repositório.
2. Baixe o arquivo `.apk` da versão mais recente.
3. No seu Android, habilite a instalação de fontes desconhecidas se solicitado.
4. Ao abrir, conceda a permissão de **"Sobrepor a outros apps"** para que o cronômetro possa flutuar.
